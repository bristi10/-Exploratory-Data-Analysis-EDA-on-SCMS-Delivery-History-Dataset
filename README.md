# -Exploratory-Data-Analysis-EDA-on-SCMS-Delivery-History-Dataset

## 📌 Project Overview

This project performs **Exploratory Data Analysis (EDA)** on the **SCMS (Supply Chain Management System) Delivery History Dataset**. The objective is to understand shipment patterns, identify data quality issues, analyze delivery performance, and uncover meaningful insights that can support supply chain decision-making.

The analysis is performed using Python and popular data analysis libraries such as Pandas, NumPy, Matplotlib, and Seaborn.

---

## 🎯 Objectives

- Explore the structure of the dataset.
- Clean and preprocess the data.
- Identify missing values and duplicates.
- Analyze numerical and categorical features.
- Visualize shipment trends.
- Examine vendor and country distributions.
- Analyze shipment costs and delivery performance.
- Generate business insights through data visualization.

---

## 📂 Dataset

**Dataset Name:** SCMS Delivery History Dataset

The dataset contains historical records of deliveries, including:

- Country
- Vendor
- Shipment Mode
- Product Group
- Freight Cost
- Line Item Value
- Scheduled Delivery Date
- Delivered Date
- Manufacturing Site
- Weight
- And other shipment-related attributes.
